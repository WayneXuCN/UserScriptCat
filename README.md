# Tampermonkey Userscripts

浏览器自动化脚本集合。

## 脚本列表

| 脚本 | 说明 |
|------|------|
| [bingReward](./bingReward) | Microsoft Rewards 自动签到 |

## 使用方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击脚本链接或复制 `.js` 文件内容到 Tampermonkey
3. 根据脚本说明进行配置
